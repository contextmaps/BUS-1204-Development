Learning Objectives

By the end of this agent segment, students will be able to:

  1. Run a domain task through their team's shared grounded agent and observe which grounding sources are active.
  2. Identify moments where the agent's behavior is driven by instructions grounding, knowledge-source grounding, or conversational grounding — and moments where no grounding source is clearly responsible.
  3. Deliberately change one grounding source mid-session and observe how the agent's behavior shifts.
  4. Articulate the priority observation about how grounding shaped their team agent's output in this session.
